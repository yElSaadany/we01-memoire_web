Les sites vitrines

Définition : Qu'est-ce qu'un site vitrine ?

Le mot vitrine fait directement le parallèle à une vitrine d'une boutique réelle, elle permet de montrer les produits que la boutique propose pour ses clients ainsi que leurs prix.

Un site dit vitrine est alors un site web qui expose ce que propose différents partis comme des PME, des artisans ou des entreprises de services.

Le site vitrine détaille alors l'histoire de celles-ci, leurs produits, leurs offres. Il est également très utile pour les artisans qui souhaitent présenter leurs réalisations.

Exemple : Site vitrine de BenjaminCode

Vous avez pu voir une photo du site vitrine du vidéaste BenjaminCode qu'il a réalisé lui même benjamincode.tv

On peut ainsi voir que l'entreprise est BenjaminCode, ses produits sont ses vidéos YouTube ainsi que ses compétences dans le web.

Nous avions aussi annoncé que le prix était sensé apparaître sur le site vitrine. Dans ce cas précis le prix n'est pas visible au premier coup d’œil. En effet il tire la rémunération de ses vidéos d'après les publicités qui sont dessus. Pour ses compétences en web on peut interpréter que le site renvoi plusieurs vidéos de sa chaîne YouTube où il aborde ses tarifs pour son travail d'intégrateur web et de développeur front end entre autre.

Point low-tech

Ces sites web sont très communs et suivent les tendances marketing. Ils sont parfaits pour présenter un MVP (produit minimum viable) d'une start-up et, à cet effet, il existe plein de framework/librairies CSS/JS dédiée à ça. Ils sont donc parmi les premiers sites web qui auraient besoin d'une refonte low-tech. En effet, le but d'une tel site (parfois composé d'une simple page très chargée), est de transformer les prospects en clients. Pour cela, le site doit : être agréable à voir, intégrer des outils d'analyses, être à la pointe de la recherche en expérience/interface utilisateur.
