La taille des sites web

Qu'est-ce que la taille d'un site web ?

La taille d'une page web - aussi appelé poids - correspond à la taille globale des fichiers d'une la page web reçus par le navigateur.
Ce poids est mesuré, comme pour tout fichier informatique, en octet. Ainsi pour une page web la taille est de l'ordre de grandeur de quelques méga octets[2].

Comment évaluer la taille d'une page web ?

Plus c'est léger, mieux c'est.

Le poids est un facteur très alarmant de la vitesse de chargement d'un site web. On peut aussi constater que ce poids augmente fortement d'année en année, il était en moyenne de 1.92Mo pour un site Desktop en 2015 et avoisinnait les 2.5Mo en 2018.[3] Il est aussi dit que le poids des sites mobiles a augmenté de 50% entre 2018 et 2020[4]. Google prend en compte le temps de chargements pour le référencement des sites web, on a donc tout intéret à réduire la taille de nos pages. En fin de compte, développeur ou particulier, sensibilisé à l'impact environnemental ou non, travailler sur l’optimisation d’un site Web s'avère être une tache très importante.

Diminuer le poid d'une page

Du côté developpeur

De façon générale il s'agit de minimiser la taille du code utilse à la page web mais aussi celle de toutes les ressources. Ainsi il faut optimiser toutes les images, mais aussi le poids de fichiers externes comme les CSS et JavaScript, éviter d'utiliser trop de librairies JS différentes, concaténer les fichiers, etc. - Certains designs peuvent faire des pages très lourdes. Il faut sensibiliser les graphistes, et s'assurer que le design répond aux objectifs du site.
  • Les minimiseurs (Closure Compiler de Google, Clean CSS) sont des outils qui permette de réduire automatiquement la taille du code
  • Il existe des équivalents de jQuery avec la même syntaxe mais utilisant seulement les fonctions essentielles
  • Adopter une syntaxe dite "vectorisé" qui permet de réduire la taille du CSS
  • Supprimer les descriptions non utilisées.

Du côté utilisateur

bloqueur : traker et publicité exemple : poid de la page d'acceuil du Figaro diviser par 3 [2].
