La taille des sites web

Qu'est-ce que la taille d'un site web ?

La taille d'une page web - aussi appelé poid - correspond à la taille globale du fichier correspondant spécifiquement à la page web et reçus par le navigateur.
Ce poid est mesuré, comme pour tout fichier informatique, en octet. Ainsi pour une page web la taille est de l'ordre de grandeur de quelques méga octets[2].
